### Thomas's Online Portfolio
Repository containing the portfolio's source code.

***

#### **Tool(s) used**:

- Vue JS (CLI)
- Vuex
- Boostrap Vue

***

### **Requirements** (Linux)

- Apache server
- NPM

***

### **Installation** (Linux)

Change the config/prod.env.js settings to match your setup.

```npm i && npm run build```

Public files are stored in dist/
